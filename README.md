# Titanic Data Analysis

This project is part of my Data Science with Python Internship at MainCrafts Technology.

## Objective

Analyze the Titanic dataset using Python to identify survival patterns based on gender, passenger class, and age.

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Survival Analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset

Titanic Dataset (Kaggle)

## Results

- Female passengers had a higher survival rate.
- First Class passengers survived more often.
- Children had better survival chances.
- Most passengers were between 20–40 years old.
